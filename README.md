### Weather IOS/Android application with React Native

- [x] Loading 화면
- [x] Weather API 사용하여 현재 위치의 날씨 불러오기
- [x] 날씨에 따른 날씨 아이콘 추가
- [x] 날씨에 대한 글귀 추가
